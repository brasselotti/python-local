# Python - local 

## 📋 Resumo
- Este repositório é dedicado ao exercício do módulo 9 (Python: Scripting), dentro da sessão "Bônus: Introdução ao Python" do curso **Desenvolvedor Full Stack Python** da EBAC.
- Neste módulo pude trabalhar com as habilidades de programação Python para a execução de código de forma local, desde a instalação e importação de bibliotecas, a criação e execução de scripts, consumo de dados externos para finalmente gerar arquivos-relatórios.
- O script deste projeto consome dados do site do BCB, trazendo para o código os valores da taxa CDI. Posteriormente, após uma validação da requisição, simulamos uma variação aleatória com o intuito de dinamizar o valor da taxa em questão, visto que o seu valor não se altera com a frequência desejada para a experimentação. Por fim, com os dados obtidos, é gerado um arquivo csv que é utilizado para plotar um gráfico que mostra a variação da taxa.
- A cada execução do script, são adicionados 10 resultados aleatórios da taxa.
